INTERNSHIP TRACKS

Game Development Internship

3-6 Months

What You'll Learn

  • Unity 3D development and C# programming
  • Game design principles and mechanics
  • 3D graphics, animations, and physics systems
  • Mobile game optimization and publishing
  • Version control with Git and collaborative development

Requirements

  • Currently pursuing or recently completed Computer Science, Game Development, or related degree
  • Basic understanding of programming concepts
  • Passion for game development and interactive media
  • Strong problem-solving skills and attention to detail

VR/AR Development Internship

3-6 Months

What You'll Learn

  • Virtual and Augmented Reality development platforms
  • Immersive user interface design
  • 3D spatial interactions and hand tracking
  • Performance optimization for VR/AR devices
  • Cross-platform VR/AR deployment

Requirements

  • Background in Computer Science, Software Engineering, or related field
  • Interest in emerging technologies and immersive experiences
  • Basic knowledge of 3D mathematics and graphics programming
  • Creative mindset and innovative thinking

UI/UX Design Internship

3-4 Months

What You'll Learn

  • User experience design for games and applications
  • Interface design principles and best practices
  • Prototyping tools and design software
  • User research and usability testing
  • Design systems and component libraries

Requirements

  • Studying Graphic Design, Interaction Design, or related field
  • Portfolio demonstrating design skills and creativity
  • Familiarity with design tools (Figma, Adobe Creative Suite, etc.)
  • Understanding of user-centered design principles
